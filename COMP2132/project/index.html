Dice Game
?

The user and the computer each roll a pair of dice 3 times. The player with the highest score wins.

  • If any die shows a 1, the score for that round is 0.
  • If both dice show the same number, the score is doubled.
  • Otherwise, the score is the sum of the two dice.
